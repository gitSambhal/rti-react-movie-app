
# React Movie App

React app to list the movies.

Steps to run 


## Run Locally

Clone the project

```bash
  git clone https://github.com/gitSambhal/rti-react-movie-app.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the app

```bash
  npm run start
```

Open the app in the browser

```bash
  http://localhost:3000/
```
